# VidGet Pro · Desktop Host

[简中](#简中) · [English](#english)

> VidGet Pro Chrome 扩展的配套桌面客户端, 接收扩展嗅探到的下载任务并落盘.
> Desktop host companion for the VidGet Pro Chrome extension — handles the actual downloads from sniffed media.

---

## 简中

### 📥 下载

到 [Releases 页](https://github.com/sfss5362/vidget-host-releases/releases/latest) 拿最新版:

| 平台 | 文件 |
|---|---|
| macOS (Apple Silicon) | `vidget-host_x.x.x_aarch64.dmg` |
| Windows x64 | `vidget-host_x.x.x_x64-setup.exe` |

### ⚙️ 安装

- **macOS**: 双击 dmg, 拖到 Applications. 首次启动若被拦截 → *系统设置 → 隐私与安全性* 点 **"仍要打开"** (一次性, 后续启动不再提示).
- **Windows**: 双击 exe, SmartScreen 提示 "未识别的发布者" → **"更多信息" → "仍要运行"**, 按引导完成.

### 🧩 配套 Chrome 扩展

装好桌面客户端后还要装 Chrome 扩展才能用:

**[📦 从 Chrome 商店一键安装](https://chromewebstore.google.com/detail/vidget-pro-download-helpe/cindnibcpmhfdaccocabjnjigcaodadk)** (推荐)

打不开商店? 桌面客户端首次启动会弹"安装指南"页, 按 3 步离线加载.

### 🌐 支持站点

- **YouTube** — [youtube.com](https://youtube.com)
- **X (Twitter)** — [x.com](https://x.com)
- **爱壹帆** — [iyf.tv](https://iyf.tv)

### 🗑 卸载

- macOS: `VidGet Pro.app` 拖到废纸篓
- Windows: 控制面板 → 程序与功能 → 卸载

## English

### 📥 Download

Grab the latest from the [Releases page](https://github.com/sfss5362/vidget-host-releases/releases/latest):

| Platform | File |
|---|---|
| macOS (Apple Silicon) | `vidget-host_x.x.x_aarch64.dmg` |
| Windows x64 | `vidget-host_x.x.x_x64-setup.exe` |

### ⚙️ Install

- **macOS**: Double-click the dmg, drag to Applications. If first launch is blocked → *System Settings → Privacy & Security* and click **"Open Anyway"** (one-time, subsequent launches won't prompt).
- **Windows**: Double-click the exe, SmartScreen warns "Unrecognized publisher" → **"More info" → "Run anyway"**, follow the wizard.

### 🧩 Companion Chrome Extension

The desktop host alone does nothing — you also need the Chrome extension:

**[📦 Install from Chrome Web Store](https://chromewebstore.google.com/detail/vidget-pro-download-helpe/cindnibcpmhfdaccocabjnjigcaodadk)** (recommended)

Store blocked in your region? The desktop client opens a "Setup Guide" page on first launch with a 3-step offline install path.

### 🌐 Supported Sites

- **YouTube** — [youtube.com](https://youtube.com)
- **X (Twitter)** — [x.com](https://x.com)
- **iyf** — [iyf.tv](https://iyf.tv)

### 🗑 Uninstall

- macOS: Drag `VidGet Pro.app` to the Trash
- Windows: Control Panel → Programs and Features → Uninstall
